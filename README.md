## Setup project
- clone https://github.com/yarina2917/project-app-server

- install server side dependencies
    
    npm install
    
- install client side dependencies
    
    cd ./admin
    
    npm install
    
## Start project   

- start client and server sides at one time

  npm run start-all

- start server side
  
  npm start  
    
- start client side
    
  cd ./admin
  
  npm start  
  
- navigate to http://localhost:4200/
